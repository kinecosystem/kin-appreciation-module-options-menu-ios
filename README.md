# Kin Appreciation Module Options Menu for iOS

[![Swift Version](https://img.shields.io/badge/Swift-5.0.x-orange.svg)](https://swift.org)
[![Platforms](https://img.shields.io/cocoapods/p/KinAppreciationModuleOptionsMenu.svg)](https://cocoapods.org/pods/KinAppreciationModuleOptionsMenu)
[![CocoaPods](https://img.shields.io/cocoapods/v/KinAppreciationModuleOptionsMenu.svg?color=6f41e8)](https://cocoapods.org/pods/KinAppreciationModuleOptionsMenu)

This module provides an interface for users to show their appreciation through gifting Kin. The module is independent from the Kin SDK, however it provides Kin branding. The UI presents a half-screen view with several gifting amount options. After choosing an option, a thank you animation will play before dismissing the view. [Read more.](https://kinecosystem.github.io/kin-website-docs/docs/documentation/appreciation-module-options-menu-ios)
